## GitHub Classroom Assignment Structure

GitHub Classroom creates student repositories with the following naming pattern:
```
georgetown-university-dsan6725-applied-genai-for-ai-developers-spring-2025-{ASSIGNMENT_ID}-{STUDENT_USERNAME}
```

For example:
- Lab 01: `georgetown-university-dsan6725-applied-genai-for-ai-developers-spring-2025-lab01-redditgpt-{username}`
- Assignment 02: `georgetown-university-dsan6725-applied-genai-for-ai-developers-spring-2025-a02-prompt-engineering-{username}`

The "GitHub Classroom Assignment ID" column above shows the `{ASSIGNMENT_ID}` portion used when creating these assignments in GitHub Classroom.
